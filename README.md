# CarCare Landing

Simple Vite + React + Tailwind landing page for CarCare, including Google Play
links for Privacy Policy and account/data deletion.

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

Deploy the generated `dist` folder to GitHub Pages or any static host.

## Public URLs

After deployment, Google Play can use separate URLs:

- `/privacy-policy/`
- `/delete-account/`
